# 小玲の運動系列選品 — Zero's

一頁式運動服飾購物表單,編輯設計風格、RWD 自適應。

## 商品
- 撞色滾邊運動背心(5 色 / S–XL / 正背面照)
- 運動上衣(2 色 / XS–XL)
- 運動短褲 款式 A / 款式 B(黑色 / S–XL)
- 滿 NT$ 2,200 加贈運動髮帶(黑色 / 深灰)

## 功能
- 顏色 / 尺寸 / 數量 多變體加入購物車
- 自動計算多件折扣與運費
- 滿額贈解鎖機制(進度條提示)
- 完整尺寸對照表 + 試穿筆記

## 本機預覽

```bash
npm install
npm start
```

開啟 http://localhost:3000

## 部署
本專案以 Railway 部署。連接 GitHub repo 後自動使用 Nixpacks 建置。
